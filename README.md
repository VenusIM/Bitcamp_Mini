# BITCAMP_MINI
비트캠프 교육과정 중에 진행한 미니 프로젝트입니다.

교육 과정중 11회의 리팩토링을 통해 완성된 쇼핑몰 프로젝트입니다.
간단한 상품관리, 구매관리 및 기타 기능들을 구현하였습니다.
